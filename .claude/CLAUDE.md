# CLAUDE.md

This file provides guidance to Claude Code when working with code in this repository. These rules are immutable and apply to every session.

Rules are organized in `.claude/rules/`:

- `project-overview.md` — Project overview, dev environment, subscription tiers, intake system, device controls
- `directory-structure.md` — Directory tree and key structural decisions
- `code-style.md` — General coding rules, docstrings, comments, section headings
- `frontend/styles.md` — CSS conventions and required root variables
