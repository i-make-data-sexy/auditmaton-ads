# Auditmaton: Ads

Auditmaton: Ads is a guided, step-by-step audit tool for advertising accounts. Practitioners work through a structured audit framework across demand-side platforms (Google Ads, Meta Ads, TikTok Ads, Microsoft Ads) and supply-side platforms (Google Ad Manager, AdSense, header-bidding setups) using each platform's own UI as the data source. There is no file upload, no API connection, and no OAuth flow in v1. The human stays in the loop at every step.

Built by Annie Cushing of Annielytics on the same Flask foundation as Auditmaton: Site and Auditmaton: Tag Management, sharing the same -ate audit framework (Educate, Investigate, Evaluate, Generate), two-device access controls, Base/Viz/AI subscription tiers, and Authorize.net billing. The taxonomy covers both the demand side (advertisers buying media) and the supply side (publishers monetizing traffic), with intake capturing the practitioner's role and platform.
